# Board : Demo dashboard

## Live link

- https://dashboard-demo-bijoy.netlify.app/

## Tech

- React.js
- Javascript
- Tailwind CSS

## Run Locally

Clone the project

```bash
  git clone https://github.com/Bijoy-007/Board
```

Go to the project directory

```bash
  cd Board
```

Install dependencies

```bash
  npm install
```

Add .env in the root

```
  REACT_APP_GOOGLE_CLIENT_ID=<Your key>
```

Start the server

```bash
  npm start
```
